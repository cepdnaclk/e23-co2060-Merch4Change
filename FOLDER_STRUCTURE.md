# Folder Structure

src/

components/
Reusable UI components.

Example:
Sidebar.jsx
Navbar.jsx
ProductCard.jsx

pages/
Main application pages.

Example:
Home.jsx
Profile.jsx
Shop.jsx

assets/
Images, icons, and static files.
global images:
    assets/logo.png
local images:
    assets/sidebar_icons/icon_name.svg

Example:
logo.png
test.jpg

services/
API communication.

Example:
authService.js
productService.js

hooks/
Custom React hooks.

Example:
useAuth.js
useFetch.js
